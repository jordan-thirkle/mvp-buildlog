# BuildLog

> **Build in public. Automatically.**

BuildLog is a build-in-public dashboard for indie makers. Log your build activity, auto-generate MDX devlog entries, draft X/Twitter posts, and export your timeline — all in one place. **Free tier, no signup, local-first.**

## Features

- **Log Activity** — Track every build, deploy, milestone, and learning with one-click entry. All data stays in your browser (local storage).
- **Generate Devlogs** — Auto-generate MDX devlog entries from your activity, ready to paste into your Astro or Next.js site.
- **Draft X Posts** — Each entry becomes a tweet. Groups become threads. Consistent cadence, zero effort.
- **Export** — Full JSON and Markdown export of your build timeline.

## Getting Started

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:4321`.

## Build

```bash
npm run build
npm run preview
```
